PROYECTO 4
Este proyecto consistió en la realización de un algoritmo genético, con y sin elitismo, para establecer 
la mejor ruta posible en términos de la menor distancia recorrida desde la ciudad de Seattle hasta el 
regreso a ella. A continuación el aporte de cada uno:
  - Carlos Montiel: generación del algoritmo genético.
  - Juan Sánchez: implementación de elitismo y implementacion de graficos para el algoritmo.
  - Esteban Alvarado: implementación de gráficos para el algoritmo.
